---
title: Event Packages
description: Published npm packages containing event type constants and Zod payload schemas.
---

All Kairosis event packages are published under the `@kairosis` scope and versioned independently.

## Packages

| Package | Description |
|---|---|
| [`@kairosis/browser-events`](/events/browser/) | Browser activity — page visits, tab events |
| [`@kairosis/calendar-events`](/events/calendar/) | Calendar events — created, updated, deleted |
| [`@kairosis/events-core`](/events/core/) | Core envelope — NormalizedEvent, Actor, Subject |
| [`@kairosis/email-events`](/events/email/) | Email events — received, sent, read |
| [`@kairosis/github-events`](/events/github/) | GitHub events — commits, PRs, issues, releases |
| [`@kairosis/health-events`](/events/health/) | Health metrics — heartbeat, sleep, recorded metrics |
| [`@kairosis/location-events`](/events/location/) | Location events — position updates, geofence transitions |
| [`@kairosis/notion-events`](/events/notion/) | Notion events — page and database changes |
| [`@kairosis/obsidian-events`](/events/obsidian/) | Obsidian events — note created, modified, deleted |
| [`@kairosis/slack-events`](/events/slack/) | Slack events — messages, reactions, channel joins |
| [`@kairosis/spotify-events`](/events/spotify/) | Spotify listening — tracks, sessions, podcasts |
| [`@kairosis/synthesized-events`](/events/synthesized/) | Synthesized events — AI-generated patterns and summaries |
| [`@kairosis/terminal-events`](/events/terminal/) | Terminal events — commands executed, directory changes |
