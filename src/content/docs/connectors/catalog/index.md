## Webhook connectors

| Connector | Description |
|---|---|
| [GitHub](/connectors/catalog/github/) | Receives GitHub webhook events — push, pull requests, issues, releases. |
| [Slack](/connectors/catalog/slack/) | Receives Slack Events API webhooks — messages, reactions, and channel joins. |

## Poller connectors

| Connector | Description |
|---|---|
| [Google Calendar](/connectors/catalog/google-calendar/) | Polls Google Calendar for new, updated, and deleted events. |
| [IMAP Email](/connectors/catalog/imap/) | Polls an IMAP mailbox for new messages and emits email.received events. |
| [Spotify](/connectors/catalog/spotify/) | Tracks your Spotify listening activity — tracks, completions, saved songs, playlists, podcasts, and listening sessions. |

## Device connectors

| Connector | Description |
|---|---|
| [Browser](/connectors/catalog/browser/) | Captures browser activity via a Chrome extension — page visits, tab opens and closes. |
| [Terminal](/connectors/catalog/terminal/) | Captures shell commands from your terminal via a shell hook. |
